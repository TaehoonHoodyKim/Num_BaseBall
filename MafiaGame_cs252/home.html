Instructions

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Number baseball game is a game based on reasoning to figure out 4 digit number randomly generated by computer.

Of course, players will not simply guess the numbers and pray that the guess will be correct.

Each time players insert numbers and submit to the computer, hints on how close the numbers are will be given.

Hints will be given in the format of baseball stats. (Strikes, Balls, Outs)

Strke : One of the digits of the number the player inserted has the the same location and the value with the digits of the number generated by computer.

Ball : One of the digits of the number the player inserted has the same value but different locations with the digits of the number generated by computer.

Out : One of the digits of the number the player inserted neither has the the same location nor the value with the digits of the number generated by computer.

The above image for an example shows 1 strike & 2 balls & 1 out

Therefore, One of the digits of the number "1234" has the same value and the location with the number generated by computer, Two of the digits of the number "1234" have same values but different locations, One of the digits of the number "1234" neither has the same value nor the location.

When the player guesses the right answer, it will show a text message saying "HomeRun!"

When the player does not guess the right number within 10 trials, the computer will show the answer and reset the number.
